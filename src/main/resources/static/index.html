PROJET FYC 2023/2024 CI/CD Groupe 5

Bienvenue sur votre première application web codée en Java. Ce projet a pour but de vous initier au déploiement applicatif, en utilisant les ressources nécessaires.

Le Pipeline a été mis en place afin de vous initier au déploiement continu, et à l'intégration du code source. Ainsi vous pouvez désormais mener vos propres projets, déployer des conteneurs grâce à l'automatisation, et créer vos propres applications en les connectant à vos bases de données.

Cette application web est basée sur un TomCat, et nous avons initié la connexion à la base de donnée PostGres pour vous apprendre à gérer vos dépendances. Cette application sert de test et d'exemple.

Nous espérons que vous avez pris du plaisir à nous suivre durant ces TPs, et vous souhaitons succès et réussite dans la suite de votre parcours. A très bientôt !
